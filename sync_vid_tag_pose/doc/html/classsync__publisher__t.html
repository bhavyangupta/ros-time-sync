Callback for approximate time message filter.

This function is called whenever the approximate time algorithm can synch messages with time stamp as close as possible. The function updates the internal "curr_frame" data member with the passed pose and april tag list. It also writes this data to a csv file with the current header timestamp

Writes current data_frames to csv file if text file output is enabled.
